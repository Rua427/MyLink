# 마이링크 (MyLink) 사용자 시나리오

## 1. 방문자 : 마이링크 페이지 방문 과정

### 페이지 접속 및 링크 이동
- 사용자는 크리에이터가 모아둔 링크 목록을 확인하기 위해 `mylink.com/displayName` 형태의 개인 공유 URL로 접속한다.
- 사용자는 원하는 목적지 웹사이트로 이동하기 위해 등록된 링크 블록을 클릭한다. (이때 시스템은 해당 링크의 클릭 카운트를 1 증가시킨다.)
- 사용자는 기존 마이링크 페이지를 유지한 채로 새로운 링크를 탐색하기 위해 새 탭(`_blank`)으로 열린 목적지 페이지를 확인한다.

---

## 2. 소유자 : 계정 관리 및 링크 관리 과정

### 회원가입 및 로그인
- 사용자는 단 한 번의 클릭으로 서비스에 가입하고 로그인하기 위해 '구글 로그인' 버튼을 클릭한다.

### 프로필 관리
- 사용자는 자신의 고유한 페이지 URL과 상단 표시 이름을 설정하기 위해 표시 이름(displayName) 영역을 클릭하여 인라인으로 수정한다.
- 사용자는 중복되지 않는 고유한 URL을 확보하기 위해 수정 시 실시간으로 제공되는 중복 검사 피드백을 확인한다.
- 사용자는 방문자에게 자신을 소개하기 위해 한 줄 소개(Bio) 영역을 클릭하여 인라인으로 수정한다.
- 사용자는 수정한 프로필 정보를 최종 저장하기 위해 우측의 체크(✅) 아이콘을 클릭한다.

### 링크 추가 과정
- 사용자는 새로운 링크를 추가하기 위해 '링크 추가' 버튼을 클릭하여 화면에 빈 링크 블록을 생성한다.
- 사용자는 새로 생성된 빈 블록에 링크 제목과 URL을 입력하기 위해 해당 영역을 클릭하여 인라인 편집 모드로 진입한다.
- 사용자는 올바른 형식의 URL을 등록하기 위해 (http/https가 누락되는 등 형식이 틀릴 경우) 붉은색 테두리 및 안내 문구 피드백을 확인하고 주소를 수정한다.
- 사용자는 입력한 링크 정보를 데이터베이스에 저장하기 위해 체크(✅) 아이콘을 클릭한다.

### 링크 수정 과정
- 사용자는 기존에 등록된 링크의 제목이나 URL을 변경하기 위해 해당 링크의 텍스트 영역을 클릭하여 인라인 편집 모드로 진입한다.
- 사용자는 수정한 내용을 최종 반영하기 위해 체크(✅) 아이콘을 클릭한다.

### 링크 삭제 과정
- 사용자는 더 이상 필요하지 않은 링크를 제거하기 위해 해당 링크 블록에 위치한 '삭제' 버튼을 클릭한다.
- 사용자는 실수로 링크가 지워지는 것을 방지하기 위해 화면에 나타난 "정말 삭제하시겠습니까?" 확인 모달 창에서 '확인' 버튼을 클릭한다.

### 링크 통계 및 공유
- 사용자는 방문자들에게 어떤 링크가 가장 인기가 있는지 파악하기 위해 각 링크 블록 하단에 표시된 누적 '클릭 수(Count)'를 확인한다.
- 사용자는 완성된 내 마이링크를 다른 소셜 미디어에 홍보하기 위해 '공유하기' (또는 주소 복사) 버튼을 클릭하여 URL을 클립보드에 복사한다.
